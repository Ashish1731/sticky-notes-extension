# 📌 Sticky Notes - Chrome Extension


> A floating sticky notes widget that stays visible on every webpage you browse. Perfect for keeping your tasks and reminders always in sight.



---

## ✨ Features

| Feature | Description |
|---------|-------------|
| 📌 **Floating Widget** | Stays visible while you browse any website |
| 🎯 **Priority Levels** | Mark tasks as High, Medium, or Low |
| 🔄 **Drag to Reorder** | Rearrange tasks by dragging the handle |
| 🔒 **Lock Mode** | Prevent accidental interaction |
| 🔆 **Opacity Control** | Adjust transparency from 10% to 100% |
| 📌 **Minimize to Pin** | Shrink widget to a movable pin icon |
| 💾 **Local Storage** | All data stays on your device |
| ⌨️ **Keyboard Shortcuts** | Quick actions for power users |

---

## ⌨️ Keyboard Shortcuts

| Shortcut | Action |
|----------|--------|
| `Ctrl+Shift+L` | Lock / Unlock widget |
| `Ctrl+Shift+N` | Hide / Show widget |
| `Ctrl+Shift+M` | Minimize to Pin / Restore |
| `Ctrl+Shift+R` | Reset all states |
| `Enter` (in input) | Add task |
| `?` | Show shortcuts help |
